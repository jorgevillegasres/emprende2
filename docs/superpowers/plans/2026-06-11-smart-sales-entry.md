# Smart Sales Entry Plan

## Goal

Make sales capture faster and more reliable by calculating revenue, cost and gross profit from the selected product and quantity.

## Steps

1. Add a pure sale totals calculator with tests.
2. Load product options when the Ventas section is active.
3. Replace the generic sales fields with a guided sales builder.
4. Keep the existing sale API payload so backend changes are not required.
5. Add focused styles for the sales summary and empty catalog state.
6. Verify unit tests, typecheck, full test suite, build and browser rendering.
