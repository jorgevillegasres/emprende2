# Recipes Implementation Plan

1. Add API tests for recipe creation, listing, and missing supply validation.

4. Add in-memory and Postgres recipe persistence.

6. Add `GET /v1/recipes` and `POST /v1/recipes`.
7. Add web client types/helpers.
8. Add a `Recetas` navigation section and compact recipe UI.
9. Run focused tests, full tests, typecheck, build, and browser layout checks.
